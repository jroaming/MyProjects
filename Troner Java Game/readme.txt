Para comenzar a jugar:
0) Ejecutar el script de la base de datos
1) Ejecutar proyecto TronerServidor
2) Pulsar CONFIGURACIÓN
3) Modificar puerto o iniciarlo directamente
4) Ejecutar proyectos TronerJugador (tantos como jugadores se quiera)
5) Escribir IP y PUERTO, y ACCEDER
6) Iniciar sesión/Llenar los datos de REGISTRO/LOGIN
7) Una vez en el menú principal, podemos pulsar cualquier opción.
8) Para empezar a jugar, pulsar o VERSUS1 o VERSUS4 (torneo no implementado del todo-somos 3 en el grupo)
9) Esperar a que encuentre partida
